# ShopFlow

ShopFlow is a production-style backend showcase demonstrating modern backend engineering concepts using Go.

## Goal

To implement every major backend concept at least once using clean, production-quality code.

## Tech Stack

- **Language**: Go
- **Database**: PostgreSQL
- **Cache**: Redis
- **Authentication**: JWT
- **Containerization**: Docker & Docker Compose
- **Architecture**: Clean Architecture / Repository Pattern / Service Layer
- **Concurrency**: Goroutines, Channels, Worker Pools
- **Background Processing**: Event-Driven Architecture, Background Jobs
- **Testing**: Unit Tests, Integration Tests
